Variable : es un contenedor donde guardamos datos, o un espacio de memoria los cuales estan asociados a una direccion de memoria. Es un conjunto de transistores que guardan distintos estados de energia. (transistores y capacitores)
    Una var esta asociada a una direccion de memoria que esta dentro de la RAM.

Cuando creo una var, el sistema operativo agrupa un conjunto de celdas con binarios dentro y guarda un dato.(ej: el nombre de una persona) y tiene adentro las respectivas direcciones de memoria. Cada celda que se junto, tiene una direccion de memoria y yo tomo solo la primera del conjunto cuando declaro una variable para llamarla.

La mutabilidad de las variables. 

Python --> 2 tipos de variables: las mutables y la inmutables.
Todas las que declaramos recien son inmutables: numero, bandera, altura, etc.
Inmutables porque no pueden cambiar su valor dentro de la memoria. 
    Esto significa que se reservan NUEVOS ESPACIOS DE MEMORIA pero los anteriores que se utilizaron vuelven a quedar disponibles. (se destruye la variable anterior y se crea una nueva y por eso cambia el id.)
    No cambia porque se crea una variable NUEVA. 

Variables mutables: pueden cambiar y se pueden modificar sin que cambie el id, no se crean nuevas variables. 
    lista = [5, 9, 7, 8]

    print(id(lista))

    lista[2] = 44

    print(id(lista))

Var mutables: List, Dic, Sets.
